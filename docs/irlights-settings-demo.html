IRLights — единый экран настроек Iris

Интерактивное демо предлагаемого единого дизайна категории (бывш. «IRLite Lights») для всех 7 шейдер-паков. Кнопки кликаются, слайдеры тянутся, наведение — тултип.
Язык лейблов: оранжевый «!» на кнопке = опции сейчас нет в части паков (см. тултип)

1. Ребрендинг: IRLite → IRLights (только видимое)

  • Все видимые названия — единые: главная кнопка §6§lIRLights§r, подэкраны без префикса «IRLite»: Specular, Shadows, Toon, Volumetric, Outline (так уже сделано в Photon и IterationRP — берём за образец).
  • Внутренние ID не трогаем: дефайны IRLITE_*, GLSL-функции irlite_*, struct IrliteLight остаются — ноль правок шейдер-кода, ноль риска сломать якоря и сохранённые настройки пользователей (Iris хранит выбор по имени дефайна).
  • Ключи экранов унифицируем в один канон: screen.IRLIGHTS, IRLIGHTS_SPECULAR, IRLIGHTS_SHADOWS, IRLIGHTS_TOON, IRLIGHTS_VOLUMETRIC, IRLIGHTS_OUTLINE (ключи невидимы игроку, живут только в наших инжектах — переименование бесплатно).

2. Единая структура (то, что в демо выше)

  • columns=2 на всех экранах во всех паках (сейчас: Bliss=1, остальные 2/default).
  • Единый порядок: на каждом подэкране первым идёт тумблер фичи, дальше параметры; группы разделены пустой строкой (<empty> <empty>).
  • Единые цвета категорий (уже почти везде так): §6 главный, §b Specular, §3 Shadows, §d Volumetric, §a Toon, §e Outline.
  • Единый набор слайдеров (sliders=): все числовые опции; кнопками-циклами остаются только тумблеры и IRLITE_VL_RESOLUTION (3 значения с текстовыми лейблами).
  • Единые лейблы и тултипы EN+RU (лучшие из текущих паков, как в демо). RU-локаль сейчас есть только у Solas/Bliss — при переносе добавляем ru_RU.lang-инжект всем паками, у которых есть файл ru_RU.
  • Единые дефолты (правка #define-значений в options-блоке патчей): IRLITE_VL_STEPS 48 (было 14), IRLITE_VL_SHADOW_STRIDE 1 (было 2), IRLITE_OUTLINE включён по умолчанию, при этом IRLITE_OUTLINE_FRONT и IRLITE_OUTLINE_GLOW выключены.

4. Порядок переноса

  • Один пак = один проход по его .irlights: (1) ключи экранов → IRLIGHTS_*, (2) раскладка экранов → канон из демо, (3) lang EN → канон, (4) lang RU → добавить/канон, (5) sliders= → канон.
  • Очерёдность: Complementary (эталон) → RethinkingVoxels → BSL → Solas → Bliss → Photon → IterationRP; после каждого — diff-проверка, в конце общий ин-гейм прогон.
