3. ДОБАВИТЬ старый LocalLightOutline:
   - defines: IRLITE_OUTLINE (off), IRLITE_OUTLINE_PIXEL_SIZE 2, IRLITE_OUTLINE_STRENGTH 0.65, IRLITE_OUTLINE_FRESNEL_POWER 2.2.
   - функции irlite_outlineDepth (линеаризация глубины — ПЕР-ПАК: depthtex1+gbufferProjectionInverse generic как в IRLEngine, ЛИБО pack-specific; под старые #version — без texelFetch) + irlite_outlineFactor (4 угловых тапа ±PIXEL_SIZE, maxBehind×edgeness, smoothstep(0.10,0.22)×smoothstep(0.05,0.12)).
   - в irlite_lightSurface: + out vec3 outlineOut; contourFactor перед циклом; per-light accumulation; outlineOut ×= surfaceMult.
   - в d4-eqv call-site: + irlite_outline арг + `fragment_color += IRLITE_INTENSITY * irlite_outline * material.albedo` (albedo-tinted), guard `|| defined IRLITE_OUTLINE`.
   - properties: вернуть `[irlite_outline]` в screen.irlite, `screen.irlite_outline = IRLITE_OUTLINE IRLITE_OUTLINE_PIXEL_SIZE IRLITE_OUTLINE_STRENGTH IRLITE_OUTLINE_FRESNEL_POWER`, 3 слайдера; lang (screen/option/suffix px).
4. Синк Modification/<pack> → дев-пак в run/shaderpacks (узнать актуальную папку — Photon юзер двигал в run/shaderpacks/Photon; у других уточнить).
5. Обновить patches/<pack>.irlights под текущий Modification. ВЕРИФИКАЦИЯ (как с Photon): javac pure-классов из irl-core (IrlPatch/IrlPatchParser/IrlPatchApplier/PatchResult/PatchEngine — только java.*) + harness `Apply(patch, Shadres/Original/<pack>, out)` → `diff --strip-trailing-cr out == Modification/<pack>`. Гоча: тело `+file` без trailing newline = косметический хвостовой diff (добавлять `\n`).
