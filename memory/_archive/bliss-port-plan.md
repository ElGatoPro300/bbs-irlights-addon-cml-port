INJECT FILE: shaders/lib/irlite/irlite_lights.glsl (+file, adapted from the Solas lib). Bliss adaptation checklist:
- discriminators: surface half `#ifdef IRLITE_SURFACE_PASS` (defined by BOTH the composite1 include op and the all_translucent include op); VL march half `#ifdef IRLITE_VL_PASS` (defined by the composite2.fsh include op — the march functions only, the buffer write stays the pack's); outline half `#ifdef IRLITE_COMPOSITE_PASS` (dimensions/composite3.fsh, Phase 4.5).
- EXCISE the Solas composite-host VL UPSAMPLE block entirely (Bliss pack upsamples its own colortex0; the block's `uniform sampler2D colortex10;` would collide with Bliss's colortex10 = resourcepack skies!). Drop the IRLITE_VL_RESOLUTION option (pack's VL_RENDER_RESOLUTION governs).
- IRLITE_ACTIVE gate: no DH_*/VOXY_* defines exist here — surface gating is per-call-site instead (composite1: `if (!isDHrange)`; all_translucent: hook is in the non-DH file by construction; DH_solid/DH_translucent are NOT hooked).
- nonTerrain: composite1 hook passes the pack's native `entities || hand` bools (runtime); all_translucent hook passes compile-time `defined ENTITIES || defined BLOCKENTITIES || defined HAND`. lightMask gate semantics unchanged (cone.z: 1 = entities-only, 2 = blocks-only).
- colour: RAW LINEAR everywhere (both hooks sit on toLinear'd albedo; VL in scene HDR units) — zero colour adaptation vs the Solas source.
- positions: fragWorld = playerPos + cameraPosition with the MVI[3] bobbing term applied EXACTLY ONCE. composite1: pass the pack's feetPlayerPos AS-IS (the pack itself adds MVI[3].xyz at the top of the surface else-branch, L977 — adding it again double-shifts under bobbing, the Phase-2 gate bug); all_translucent: build our own mat3(MVI)*viewPos + MVI[3].xyz from raw viewPos (matches the pack's own L617 form; dim-independent).
- normals: composite1 normal ~world-space (verify!), all_translucent normal VIEW (use its ready worldSpaceNormal or mat3(MVI)*normal). Keep the degenerate-normal fallback.
- specular: keep the Solas self-contained world-space GGX (pack's DoSpecularReflections/GGX are sun-coupled). Spec nonTerrain-only (the BSL "незер не должен отражать" rule).
- in-file #extension SSBO idiom at #version 120 = BSL-proven. Banner comments: ASCII only, NO `*/` globs.
