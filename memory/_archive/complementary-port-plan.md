Roadmap for porting the IRLite inject to Complementary (Reimagined+Unbound unified pack), third target after IterationRP and Photon. Parent: [[MEMORY]]. WHERE/HOW to hook = [[complementary-pipeline]] (read it first every phase). Port SOURCE = the Photon Modification inject (Shadres/Modification/Photon/shaders/include/irlite/irlite_lights.glsl — has all bugfixes), NOT the older iterationrp patch. Folder/test loop = [[sync-workflow]]. Final output = patches/complementaryreimagined.irlpatch ([[patcher]]).

GROUND RULES (same as the Photon port, user-set): work entirely in Shadres/Modification/ComplementaryReimagined (clean copy of Shadres/Original/ComplementaryReimagined made 2026-06-11, 436 files verified); ALL phases hand-edit Modification; diff + build the .irlpatch ONLY at the very end (Phase 5). Each phase = its own session. The Java addon and the patcher applier are pack-agnostic and DO NOT change.

INJECT FILE: shaders/lib/irlite/irlite_lights.glsl. Self-contained IRLITE_* defines at top; in-file `#extension GL_ARB_shader_storage_buffer_object : enable` (+ texture_cube_map_array in Phase 2) — the pack's own SSBO files prove the in-file idiom at #version 130, so NO wrapper edits at all. Split: surface half `#ifndef COMPOSITE1`, VL half `#ifdef COMPOSITE1`.
