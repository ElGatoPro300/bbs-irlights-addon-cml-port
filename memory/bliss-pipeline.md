KEY DELTAS vs prior ports (the porting checklist):
- vs Photon (deferred analog): entity/hand masks are NATIVE at the shading pass (no bit-7 write/decode/mask ops); TWO surface hooks needed (deferred composite1 + forward all_translucent) — Photon never hooked translucents (open gap there); VL piggybacks the pack's own reduced-res pass exactly like Photon's c0_vl ride; per-program wrapper #extension edits NOT needed (in-file extension in the lib, Solas/BSL idiom).
- vs Solas/BSL (the lib source): colour convention IDENTICAL (raw linear everywhere — composite1 albedo toLinear'd, all_translucent Albedo toLinear'd, VL in scene HDR units); positions camera-relative player-space (+cameraPosition for absolute, MVI[3] bobbing rule); normal spaces MIXED (composite1 ~world, all_translucent VIEW + ready worldSpaceNormal) — adapt per hook; no added deferred2/wrappers/colortex10/size.buffer (whole VL host block of the lib replaced by the piggyback half); IRLITE_VL_RESOLUTION option dropped; DH gate runtime not compile-time; CRLF same as BSL/Solas (single-line anchors); #version 120 same as BSL.
- OUTLINE candidate host = dimensions/composite3.fsh (slot 4 merge pass — post-translucent scene + masks colortex7.a/colortex1 available; RENDER_SCALE-aware neighbor sampling required). Phase 4.5, verify masks reach it.
